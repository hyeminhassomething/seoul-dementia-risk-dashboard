SLIDE 04 · LIMITATION

기존 분석의 한계

자치구 평균 25km² vs 50m 격자 — 약 10,000배 해상도 차이가 만드는 사각지대

📐 공간 해상도 미스매치
  • 자치구 평균 면적 약 25km² vs 실종자 발견 반경 250m (약 0.0625km²)
  • 약 250배 공간 미스매치 (면적 환산 시 약 1만 배)
  • 자치구 단위 예산은 위험·안전 격자에 동일 분산 → 효율성 저하
🔍 사각지대 식별 불가
  • 안전구역(LL) 사이에 고립된 위험 격자 (HL 948개)는 자치구 평균에 묻혀 보이지 않음
  • 동 단위 평균 위험도 안에는 격자 단위로 5배 이상의 편차 존재
  • 자치구 평균에 가려진 동선·환경 위험을 식별할 도구 부재
💡 원인 분석 부재
  • 같은 위험 격자라도 환경 유형이 다르면 처방도 달라야 함
  • 기존 정책은 일률 처방 — 골목 미로형과 도심 상권형에 동일 CCTV 정책
  • 환경 시그너처를 식별하는 비지도 진단 도구 없음

SLIDE 10 · LISA

LISA 4-Cluster — 정의 · 근거

Local Moran's I · Anselin (1995) · 공간 자기상관 학계 표준
각 격자와 이웃의 위험도를 비교해 HH / HL / LH / LL 4가지 유형으로 분류
Queen contiguity weight + permutation test (p<0.05) 적용

Moran's I Scatter Plot — Anselin (1995)
① 채택 근거
  • 자치구 평균에 묻힌 outlier(HL) 직접 식별
    HL 948개 = 안전구역 사이 핫스팟·사각지대
  • 통계적 유의성 검정 (p<0.05)
    Queen contiguity + permutation test → 우연이 아닌 핫스팟만 추출
  • 학계 표준 — 공간 역학
    범죄·전염병·자살·교통사고 공간 클러스터 분석의 표준 방법론 (Anselin 1995)
② 핵심 가설 제안
LL에서 실종 발생 → 인접 HL에서 발견 가능성↑
HL은 「사각지대 + 발견 거점」 이중 의미
Anselin L. (1995). LISA. Geogr Anal.

  • 실종 신고 발생 시 신고 위치 반경 500m 평균 수백 대 CCTV가 검색 대상
